# Projekt-Excel
Analiza sprzedaży czekolady
Dashboard w Excelu

## Opis projektu

Projekt przedstawia analizę danych sprzedażowych firmy produkującej czekoladę. 
Celem projektu było przeanalizowanie wyników sprzedaży oraz stworzenie interaktywnego dashboardu w Microsoft Excel.

Zbiór danych zawiera ponad 3000 rekordów sprzedaży obejmujących różne kraje, produkty oraz sprzedawców.

 ## Wykorzystane narzędzia

- Microsoft Excel
- Tabele przestawne (Pivot Tables)
- Wykresy przestawne (Pivot Charts)
- Fragmentatory
- Analiza danych

## Funkcjonalności dashboardu

Dashboard umożliwia analizę:

- całkowitej sprzedaży
- sprzedaży według krajów
- sprzedaży według produktów
- najlepszych sprzedawców
- trendów sprzedaży w czasie

Zastosowano również interaktywne filtry umożliwiające analizę danych według:

- kraju
- roku

## Najważniejsze wnioski z analizy

- Największą sprzedaż generują kraje takie jak Australia, Wielka Brytania oraz Indie.
- Produkty **Smooth Silky Salty**, **50% Dark Bites** oraz **White Choc** należą do najlepiej sprzedających się produktów.
- Najlepsi sprzedawcy generują znaczną część całkowitej sprzedaży.
- Analiza danych wskazuje na różnice w poziomie sprzedaży w zależności od okresu w roku.

## Zawartość repozytorium

- plik Excel z analizą danych
- dashboard sprzedażowy
- wnioski z analizy danych
